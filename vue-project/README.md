## How to Run Locally?

Open your terminal.

First, navigate to the Vue project directory:
(If you opened the terminal in Cursor, you can use the following relative path; otherwise, use the full path.)

```sh
cd vue-project
```

Install all required dependencies:

(Run this for **initial setup** or when dependencies change.)

(This reads from `package.json`, similar to installing from `requirements.txt` in Python.)

```sh
npm install
```

> **About `package-lock.json`:** This file is automatically generated by npm to lock exact versions of all dependencies. While `package.json` often uses version ranges (like `^3.4.0`), `package-lock.json` records the precise versions installed. This ensures everyone on the team uses identical package versions, avoiding "works on my machine" issues. Always commit this file to Git.

Then start the development server:

```sh
npm run dev
```

The application will be available at `http://localhost:5173/` (or another port if 5173 is occupied).

### Daily Development Workflow

After the initial setup, your daily routine is simple:

```sh
cd vue-project
npm run dev
```

That's it! No need to run `npm install` again unless dependencies change.
